# House Price Prediction

This project predicts house prices using regression models.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Steps
- Data loading
- Basic EDA
- Preprocessing
- Model training
- Evaluation

## Models Used
- Linear Regression
- Random Forest Regressor

## Result
Random Forest performed better than Linear Regression.
